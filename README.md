# WUCHENG

WUCHENG is an AI film production project built around a quiet mystery in Gray Harbor. This repository is the production workspace for script development, storyboards, concept art, asset tracking, and delivery planning.

## Current Focus

Sprint 1 turns the existing pilot outline into a usable production package:

- lock the pilot script direction
- define the first storyboard pass
- establish visual references for Gray Harbor, Qi Zhou, and the blue dream
- create an asset index that tracks source, status, and ownership
- keep production decisions visible for future sprints

## Workspace Map

| Path | Purpose |
| --- | --- |
| `CODEX.md` | Project engineering guide for Codex work in this repo. |
| `docs/` | Roadmaps, planning notes, and production documentation. |
| `script/` | Canon script materials, outlines, scene drafts, and revisions. |
| `storyboard/` | Storyboard frames, shot lists, animatic notes, and board exports. |
| `concept_art/` | Character, environment, prop, color, and mood explorations. |
| `assets/` | Asset index, source references, prompts, renders, and review state. |
| `production/` | Production handbook, review gates, naming rules, and delivery checklists. |

## Canon Source

The current story source is `script/pilot/pilot_outline_v1.md`. Treat it as the creative baseline until a later script revision supersedes it.

## Production Rules

- Keep every production asset traceable through `assets/ASSET_INDEX.md`.
- Prefer small, reviewable commits with clear production intent.
- Do not mark AI-generated images, audio, or video as final until their source prompt, tool, rights status, and review state are recorded.
- Preserve the pilot tone: quiet, tactile, restrained, and mystery-first.
